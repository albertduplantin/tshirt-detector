========================================
DETECTION DE VETEMENTS - VERSION PORTABLE
========================================

🎯 DESCRIPTION:
Application C++ portable pour détecter automatiquement si vous portez 
un t-shirt ou un autre vêtement via votre webcam.

✅ FONCTIONNALITES:
- Détection en temps réel via webcam
- Classification automatique (t-shirt, pull, chemise, etc.)
- Interface graphique intuitive
- Sauvegarde d'images
- 100% portable (aucune installation requise)

🚀 UTILISATION RAPIDE:
1. Double-cliquez sur "detection_vetements.exe"
2. L'application s'ouvre automatiquement
3. Votre webcam se lance et la détection commence
4. L'application affiche le type de vêtement détecté

⌨️ CONTROLES:
- Q : Quitter l'application
- S : Sauvegarder une capture d'écran
- R : Redémarrer la détection

🎨 INTERFACE:
- Cadre blanc : Zone de détection
- Texte vert : T-shirt détecté
- Texte rouge : Autre vêtement détecté
- Instructions : Affichées en bas de l'écran

💾 SAUVEGARDE:
Les images sont sauvegardées dans le même dossier que l'application
avec le nom "capture_0.jpg", "capture_1.jpg", etc.

🔧 DEPANNAGE:

❌ "Impossible d'ouvrir la webcam"
→ Vérifiez que votre webcam est connectée et fonctionne
→ Fermez les autres applications qui utilisent la webcam
→ Redémarrez l'application

❌ "Aucun vêtement détecté"
→ Assurez-vous d'être bien visible dans le cadre
→ Éclairage suffisant recommandé
→ Vêtement de couleur contrastée avec l'arrière-plan

❌ "Application ne se lance pas"
→ Vérifiez que tous les fichiers sont présents
→ Essayez de lancer "Lancer_Application.bat"
→ Redémarrez votre PC si nécessaire

📁 CONTENU DU PACKAGE:
- detection_vetements.exe : Application principale
- *.dll : Bibliothèques nécessaires
- Lancer_Application.bat : Script de lancement
- README_UTILISATION.txt : Ce fichier

🌐 COMPATIBILITE:
- Windows 10/11 (64-bit)
- Aucune installation requise
- Fonctionne sur clé USB
- Pas de droits administrateur nécessaires

📞 SUPPORT:
Pour toute question ou problème, consultez le README principal
ou le fichier de documentation du projet.
